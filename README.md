# pr-submission-skill

Standalone Agent Skill for preparing GitHub issues and pull requests with
prior research, scoped framing, and maintainer-friendly PR bodies.

## Install

Copy the skill folder into your Codex skills directory:

```bash
cp -R pr-submission ~/.codex/skills/pr-submission
```

## Skill

See [`pr-submission/SKILL.md`](pr-submission/SKILL.md).
